# redTetris

## Multiplayer online tetris.

## Server

```js
npm run srv-install; npm run srv-start;
```

## Client
```js
npm run clt-install; npm run clt-start;
```

## Run test
#### server
```js
npm run srv-test
```
#### client
```js
npm run clt-test
```


### storybook
